# Glanceify ⚡

**Speed read any text on the web.** Select text and read it word-by-word using RSVP (Rapid Serial Visual Presentation) technology.

## Features

- 📖 **RSVP Speed Reading** — Words displayed one at a time for focused, fast reading
- 🎯 **ORP Highlighting** — Optimal Recognition Point highlighted for each word
- ⚡ **Adjustable Speed** — 100 to 900 words per minute
- 🖱️ **Two Ways to Trigger** — Select text and click the floating button, or right-click → "Read with Glanceify"
- ⌨️ **Keyboard Shortcuts** — Space (play/pause), arrows (navigate/speed), Esc (close)
- 🎨 **Modern Dark UI** — Clean, eye-friendly overlay design
- 📊 **Progress Tracking** — Visual progress bar and word counter

## Installation

### From Chrome Web Store
[Glanceify](https://chromewebstore.google.com/detail/glanceify/ggplcijbloabllhpbdedenhkldbhknjg)

### Manual Installation (Developer Mode)
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (top right toggle)
4. Click **"Load unpacked"**
5. Select the `fast_read_ext` folder
6. Glanceify is now active on all web pages!

## Usage

### Method 1: Floating Button
1. Select any text on a webpage
2. Click the purple **▶ Read** button that appears
3. The reading panel opens — press Play to start

### Method 2: Context Menu
1. Select any text on a webpage
2. Right-click → **"Read with Glanceify"**
3. The reading panel opens — press Play to start

### Keyboard Shortcuts (while panel is open)
| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `←` | Previous word |
| `→` | Next word |
| `↑` | Increase speed (+25 WPM) |
| `↓` | Decrease speed (-25 WPM) |
| `Esc` | Close panel |
